## User Flow

### New User

1. Lands on **Homepage** (`/`)
2. Clicks **"Join the Free Programme"**
3. Completes **Assessment** questionnaire (`/assessment`)
4. On submission, redirected to **AI Coach** (`/chat-v2`)
5. Speaks with David Haye AI Coach
6. Scrolls down to explore nicotine options:
   - IQOS Store (`/iqos`)
   - Heated Tobacco (`/heated-tobacco`)
   - Vapes (`/vapes`)
   - Nicotine Pouches (`/nicotine-pouches`)

### Returning User

- Header automatically shows **"Go to Agent Page"** button
- Takes them straight back to **AI Coach** — no need to redo the assessment
